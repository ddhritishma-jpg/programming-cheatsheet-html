Basic File Structure

The boilerplate code needed to start any HTML file.

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    Content goes here
  </body>
</html>
Explanation: Every HTML file starts with <!DOCTYPE html> and wraps content in <html>, <head>, and <body> tags.
